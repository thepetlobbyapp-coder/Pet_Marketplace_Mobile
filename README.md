# Pet Marketplace — Mobile

App de tutores e prestadores. **React Native + Expo + TypeScript**.
Android / Play Store primeiro; iOS preparado na arquitetura (publicação futura).

> Bloco 0 entrega apenas o esqueleto. A inicialização do Expo é o **Bloco 3**.

- Documentação oficial: `./docs/` (cópia sincronizada — editar na raiz).
- Agentes: `./.codex/`.
- Textos finais **somente** via i18n `en-GB`. **Sem texto hardcoded.**
- Fase 1: chat só texto; sem câmera/galeria/microfone; localização só em contexto.
- Nunca exibir endereço completo/coordenadas exatas de outros usuários.

## Scripts (placeholder até Bloco 3)

```bash
pnpm install
pnpm start    # definido no Bloco 3 (Expo)
```
