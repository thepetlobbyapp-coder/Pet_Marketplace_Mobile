# Pet Marketplace — Backend

Cérebro da aplicação. **NestJS + TypeScript + Supabase (PostgreSQL/PostGIS)**.
Autoridade de regras de negócio, RBAC, autorização, busca geográfica e auditoria.

> Bloco 0 entrega apenas o esqueleto. A inicialização do NestJS é o **Bloco 1**.

- Documentação oficial: `./docs/` (cópia sincronizada — editar na raiz do repositório).
- Agentes: `./.codex/`.
- Variáveis de ambiente: copiar `.env.example` para `.env`.
- Fase 1: **sem pagamento**, sem documentos obrigatórios de prestador.
- Nunca expor endereço completo/coordenadas exatas entre usuários.

## Scripts (placeholder até Bloco 1)

```bash
pnpm install
pnpm dev      # definido no Bloco 1 (NestJS)
```
